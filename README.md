# movie-data-analysis-python
End-to-end movie data analysis project featuring data cleaning, EDA, visualization, and insight generation using Python.

Movie Data Analysis using Python

Project Overview

In this project, I analyzed a movie dataset containing more than 9,800 records. The goal was to understand movie trends based on genres, ratings, popularity, and release years using Python.
I performed data cleaning, preprocessing, exploratory data analysis (EDA), and visualization to identify useful patterns in the dataset.

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

Dataset Information

The dataset contains movie-related information such as:

* Title
* Genre
* Popularity
* Vote Average
* Release Date
* Original Language

Total Records: 9,827+

Data Cleaning

The following steps were performed:

* Converted Release Date into Year format
* Removed unnecessary columns
* Categorized Vote Average into groups
* Split multiple genres into separate rows
* Converted Genre column into categorical datatype
* Removed missing values

Key Findings

* Drama is the most common genre in the dataset.
* Most movies belong to the Below Average vote category.
* Spider-Man: No Way Home has the highest popularity score.
* The United States vs. Billie Holiday has the lowest popularity score.
* 2020 recorded the highest number of movie releases.

Visualizations

The project includes:

* Genre Distribution
* Vote Average Distribution
* Release Year Distribution
* Popularity Analysis

Conclusion

This project helped me improve my skills in data cleaning, data analysis, and data visualization using Python. It also provided insights into movie trends and audience preferences.
